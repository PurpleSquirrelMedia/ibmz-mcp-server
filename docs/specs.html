Technical Specifications

IBM Z MCP Server v1.0.0

Environment Variables

Key Protect Configuration

Variable Required Description
IBM_CLOUD_API_KEY Yes IBM Cloud API key with access to Key Protect
KEY_PROTECT_INSTANCE_ID Yes GUID of your Key Protect service instance
KEY_PROTECT_URL No Key Protect endpoint (default: https://us-south.kms.cloud.ibm.com)

z/OS Connect Configuration

Variable Required Description
ZOS_CONNECT_URL Yes* Base URL of z/OS Connect server (e.g., https://mainframe:9443)
ZOS_CONNECT_USERNAME No Username for basic authentication
ZOS_CONNECT_PASSWORD No Password for basic authentication

* Required only if using z/OS Connect tools

Key Protect Tool Specifications

key_protect_list_keys

List encryption keys in your Key Protect instance.

Parameter Type Required Description
limit number No Maximum keys to return (default: 100)
offset number No Number of keys to skip (default: 0)
state array No Filter by state: 1=Active, 2=Suspended, 3=Deactivated, 5=Destroyed

key_protect_create_key

Create a new encryption key in the HSM.

Parameter Type Required Description
name string Yes Unique name for the key
description string No Description of key purpose
type string No "root_key" or "standard_key" (default)
extractable boolean No Allow key material export (default: false)
payload string No Base64 key material for BYOK

key_protect_wrap_key

Wrap (encrypt) a data encryption key with a root key.

Parameter Type Required Description
key_id string Yes ID of the root key
plaintext string Yes Base64-encoded DEK to wrap
aad array No Additional authenticated data

key_protect_unwrap_key

Unwrap (decrypt) a wrapped data encryption key.

Parameter Type Required Description
key_id string Yes ID of the root key
ciphertext string Yes Base64-encoded wrapped DEK
aad array No AAD (must match wrap operation)

key_protect_rotate_key

Rotate a root key to a new version.

Parameter Type Required Description
key_id string Yes ID of the root key to rotate
payload string No New key material for BYOK rotation

key_protect_delete_key

WARNING: Key deletion is irreversible. Data encrypted with deleted keys becomes permanently unrecoverable.

Parameter Type Required Description
key_id string Yes ID of the key to delete
force boolean No Force delete even with associated resources

z/OS Connect Tool Specifications

Note: z/OS Connect requires access to an IBM mainframe with z/OS Connect EE installed. These tools will return configuration errors if ZOS_CONNECT_URL is not set.

zos_connect_list_services

List all deployed z/OS Connect services.

No parameters required.

zos_connect_call_service

Call a z/OS Connect service to interact with mainframe programs.

Parameter Type Required Description
service_name string Yes Name of the z/OS Connect service
operation string No API operation path (default: "/")
method string No HTTP method: GET, POST, PUT, DELETE, PATCH
payload object No JSON request body
path_params object No Path parameter substitutions
query_params object No Query string parameters

Security Considerations

Key Protect

  • Keys are stored in FIPS 140-2 Level 3 certified HSMs
  • Root keys never leave the HSM hardware
  • All API calls use TLS 1.2+ encryption
  • IAM authentication required for all operations
  • Activity Tracker logs all key operations

z/OS Connect

  • Supports TLS/SSL for transport security
  • Basic authentication or SAF-based security
  • Credentials should be stored securely (not in code)
  • Network access typically requires VPN or private connection

Error Handling

All tools return structured error messages:

{
  "content": [{
    "type": "text",
    "text": "Error: Key Protect not configured. Set IBM_CLOUD_API_KEY and KEY_PROTECT_INSTANCE_ID environment variables."
  }]
}
